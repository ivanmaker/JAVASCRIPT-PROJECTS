### Tic Tac Toe Game <a name="tic-tac-toe-game"></a>
Classic Tic-Tac-Toe game that uses a random number to choose where the computer player places an O. When one player wins a line is drawn through the three Xs or Os that constituted the win and confetti is displayed briefly before the board is reset.

### Calculator <a name="calculator"></a>
A simple calculator that uses addition, subtraction, multiplication, and division in a webapp

### Pizza Order Menu <a name="pizza-order-menu"></a>
A simple site that uses radio buttons and check boxes to allow the user to select a pizza order. The cost of the order is tallied and displayed on the screen after hitting Place Order

### AJAX <a name="ajax"></a>
A very straightforward project focusing on the basics of AJAX to push text from one HTML file to another using a Apache server.

### jQuery <a name="jquery"></a>
Uses JS and jQuery to demonstrate shifting elements around, animating a shine effect, and dropdown toggles

### Simple To-Do App <a name="simple-todo-app"></a>
Uses localStorage to stage To-Do tasks in browser memory so they can be added and removed from an array
